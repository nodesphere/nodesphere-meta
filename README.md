# Nodesphere Meta

Meta interface to various Nodesphere interfaces.

Demo: http://meta.nodesphere.org/

## Development

### Setup

```
npm install -g grunt-cli bower
npm install
bower install
```

### Starting a Local server

```
grunt serve
```

### Building a distribution

```
grunt build
```
